# 🛋️ Cantinho Decor: Your Home, Your Style!

## Project Overview

Welcome to **Cantinho Decor**, an online platform designed to help users find furniture and decoration items to transform their homes.

The project was developed to provide an intuitive and modern experience for browsing and exploring home decor products. The goal is to make it easier for users to discover items and visualize how they can improve their living spaces.

Whether someone is looking for their first decorative piece or planning a complete room makeover, Cantinho Decor offers a simple and organized way to explore a catalog of products.

This project was developed as part of a **Computer Science academic assignment** and demonstrates the structure of a **full-stack web application**.

---

# Technologies Used

The project was built using the following technologies:

- **React** – Front-end user interface
- **Node.js** – Backend server and API structure
- **JavaScript** – Application logic
- **HTML** – Page structure
- **CSS** – Styling and layout

---

# APIs

The system was designed using dedicated APIs to organize the platform structure.

### User Registration and Login API
Responsible for handling user registration and authentication, ensuring secure access to the platform.

### Product API
Manages the product catalog, allowing users to view, add, and manage decoration items available on the platform.

The main goal of Cantinho Decor is to create a **modern and user-friendly platform** for discovering furniture and decoration products.

The platform includes the following functionalities:

# How to Run the Project

To run Cantinho Decor locally, follow the steps below.

## 1️⃣ Prepare the Environment

Make sure you have **Node.js** and **npm** installed.

You can download Node.js here:

https://nodejs.org/

Node.js installation already includes npm.

---

## 2️⃣ Verify the Installation

Open your terminal and run:
node -v
npm -v


If both commands return a version number, your environment is ready.

---

## 3️⃣ Install Dependencies

Navigate to the project folder and run:
npm install


---

## 4️⃣ Run the Project

Start the development server:
npm start

---

Note: Some APIs used in the project may no longer be active since this project was developed for academic purposes.
